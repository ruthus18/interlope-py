# Interlope Game Engine (Python Edition)

**Features:**

* C-like modules
* Scene and world management through local database


### Requirements

* Python 3.12
* Poetry
* OpenGL lib (check `pyopengl` setup)
* PIL


### Installation

```bash
poetry install
```

### Launch

**Linux**
```bash
./run.sh
```

**Windows**
```sh
./run.bat
```

### Controls

~ - Switch between camera and cursor control
